Usando link externo

Você pode acessar meu repositório no github.

Para que o link abra em outra página usamos na TAG <a> a marcação target="_blank" - Para abrir em uma página em branco, e podemos usar o rel="external" que mostra ao código que é uma relação com código externo.

Você também pode acessar meu linkedin.

Link Interno

Aqui a gente coloca o link para uma outra página dentro do nosso site. Há dois jeitos de fazer isso.

  1. Criando um novo arquivo, usando o href="seu_arquivo.html" e linkar.
    • Assim: Clica aqui
  2. Outro meio é linkar essa página a outra dentro de uma outra pasta, usando o caminho até ela. Ex:
    • Clicando aqui, você vai abrir a aula do cap 09
